# Portfolio de Billal Mechekour

Portfolio React moderne construit avec Vite.

## Démarrage

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
npm run preview
```

## Personnalisation rapide

- Données principales: `src/data/portfolioData.js`
- Design et animations: `src/App.css`
- Structure globale: `src/App.jsx`

Certains dépôts GitHub n'ont pas été fournis en URL directe. Ils pointent actuellement vers ton profil GitHub global et peuvent être remplacés facilement dans `src/data/portfolioData.js`.
